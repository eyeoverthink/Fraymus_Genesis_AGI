# Fraymus Genesis & AGI

Genesis framework and AGI applications.

## Applications

- **GenesisGPT** - Genesis GPT - Self-Modifying AI
- **GenesisPrime** - Genesis Prime - The Architect
- **GenesisRuntime** - Genesis Runtime - Execution Engine
- **GenesisZero** - Genesis Zero - The Void
- **GenesisBlock** - Genesis Block - Blockchain Core
- **FractalOptimizer** - Fractal Optimizer - Self-Optimizing Code
- **GitCortex** - Git Cortex - Version Control Intelligence
- **Hippocampus** - Hippocampus - Memory Formation
- **LibraryEater** - Library Eater - Code Consumption
- **Matrix** - Matrix - Digital Reality
- **OllamaSpine** - Ollama Spine - LLM Integration
- **Optimizer** - Optimizer - Performance Tuning
- **Singularity** - Singularity - The Event Horizon
- **SystemMain** - System Main - Entry Point
- **Tesseract** - Tesseract - 4D Computing
- **TheArchitect** - The Architect - Design Intelligence
- **Tokenizer** - Tokenizer - Language Processing
- **Trainer** - Trainer - Model Training
- **TransformerBlock** - Transformer Block - Neural Architecture
- **Transmudder** - Transmudder - Code Transformation

## Dependencies

- LWJGL 3.3.3 (OpenGL, GLFW, STB)
- ImGui Java 1.86.11
- JOML 1.10.5
- Ollama4j 1.1.0
- Neural network libraries

## Build

```bash
cd Asset-Manager
./gradlew build
```

## Run Genesis & AGI Apps

```bash
./gradlew runGenesisGPT          # Run Genesis GPT
./gradlew runGenesisPrime        # Run Genesis Prime
./gradlew runGenesisRuntime      # Run Genesis Runtime
./gradlew runGenesisZero         # Run Genesis Zero
./gradlew runGenesisBlock        # Run Genesis Block
./gradlew runFractalOptimizer    # Run Fractal Optimizer
./gradlew runGitCortex           # Run Git Cortex
./gradlew runHippocampus         # Run Hippocampus

./gradlew runMatrix              # Run Matrix
./gradlew runOllamaSpine         # Run Ollama Spine
```
